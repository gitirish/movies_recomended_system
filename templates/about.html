About the Project

This project is a movie recommendation system that suggests movies based on user preferences. It uses data from The Movie Database (TMDB) website, both the dataset and the API, to provide accurate recommendations.

Tools Used

  • Flask: A micro web framework for Python.
  • TMDB API: Used to fetch movie data.
  • TMDB Dataset: Provides comprehensive movie data for model training.
  • Machine Learning Algorithms: Used to build the recommendation system.

Methodology

The recommendation system is built using the following methodologies:

  • Cosine Similarity: Measures the cosine of the angle between two non-zero vectors of an inner product space.
  • Count Vectorizer: Converts a collection of text documents to a matrix of token counts.
  • Porter Stemmer: A process for removing the commoner morphological and inflexional endings from words in English.
